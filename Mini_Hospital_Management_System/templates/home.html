Core Features

Why Choose Mini HMS?

Built with efficiency and security in mind. Our system handles role-based access for doctors and patients seamlessly.

Smart Appointments

Real-time slot blocking ensures no two patients can book the same doctor at the same time. The system prevents race conditions instantly.

Calendar Sync

Automatically adds confirmed appointments to both the Doctor's and Patient's Google Calendar using OAuth2 API integration.

Serverless Alerts

Powered by AWS Lambda. Patients receive instant BOOKING_CONFIRMATION emails, and new users get a SIGNUP_WELCOME.

For Doctors

  • Dashboard: Manage personal availability and view upcoming schedules.
  • Slot Management: Create specific time slots (e.g., 10:00-10:30) easily.
  • Privacy: View and manage only your own bookings securely.

For Patients

  • Search: Browse doctors and view their specific available time slots.
  • Instant Booking: Secure a slot with a single click.
  • Sync: Get automatic calendar invites and email confirmations.
